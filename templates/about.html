Language summary

Either fill the dialogue to be summarised into the text field or upload a text file using the "Browse" button. The text file must have the following structure:

If the dialog is already translated to english, you can check the checkbox "already translated". This prevents the paid API request for the translation.

In the next input form, you can choose into how much words, the text is split for each pass of the summarizing model. This changes the length and the accuracy of the resulting summary.

At the end, you can specify the names of the advisor and the customer. This leads to a better result because the model looses all its context information after each pass including the names.
